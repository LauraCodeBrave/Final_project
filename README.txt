Inkstellation:

*Description*:

_ This project is a website called "Inkstellation" . It's a poetry website where poetry lovers find everything they need to enjoy reading and writing poetries.This website contains famous poetries that users can read or listen to instead.It also explores famous poetes.The website also allows users to write poetries and save them. The website's purpose is providing users with a fun and creative way to enjoy thir hobby throughtout the features it offers.

*Technologies Used*:

_HTML for the structure and content
_CSS for style and responsiveness
_JavaScript for functions

*local storage usage*:

_Used to save the font selected by the user so it doesn't go back to default after refreshing
_Used to save the poetry of the day so it stays the same until the date changes 
_Used to save the poetries after the user writes them and clicks on the save button that has an ink pen


*Api usage*:

_Used an api in the main page to get random poems for the poem of the day envelope
_Used api to get words that rhyme to the word the user entered

How to use it:

_Open index.html in a web browser to explore the website.
_Explore the poetry books you're interested in and famous poets.
_Read the poem of the day on the main page.
_click on Reading to read poetry books or listen to them and track your mood.
_Go to writing to write your own poetry and save it and search for words that rhyme to help you with it.
_You can also go to constellation to explore the sky of stars where each star holds a different quote.


*Features*:

_Font Changer: users can change the font according to their liking.
_Poetry Gallery: Explore famous poetries and poets with a poem of the day.
_Audio Books: Able to read or listen to books.
_Mood Tracker: Track users mood throughout the week.
_Writing poetries: a place for users to write poetries and save them.
_Rhyme Side bar: a side bar where users can search for words that rhyme.
_Constellation: A sky of stars that hold quotes.
